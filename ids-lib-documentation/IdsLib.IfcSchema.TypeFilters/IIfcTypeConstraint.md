# IIfcTypeConstraint interface

Represents a constraint on the types of an entity, such as the allowed types for an attribute or the types of entities that can be related in a relationship. The constraint is defined as a set of concrete types, represented as upper invariant strings. The interface provides methods for intersecting and unioning constraints, as well as checking if a constraint is empty (i.e., has no allowed types). The standard empty constraint is the static [`Empty`](./IfcTypeConcreteListConstraint/Empty.md).
