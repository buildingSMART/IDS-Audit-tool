# SchemaInfo.TrySimplifyTopClasses method

Attempts to identify the minimal set of top-level classes that collectively cover all specified concrete class names within the given schema versions.

```csharp
public static bool TrySimplifyTopClasses(IEnumerable<string> concreteClassNames, 
    IfcSchemaVersions schemas, out IEnumerable<string> topClasses)
```

| parameter | description |
| --- | --- |
| concreteClassNames | A collection of concrete class names to be covered by the resulting top-level classes. Class names are compared case-insensitively. |
| schemas | The schema versions to use when determining class hierarchies and coverage. |
| topClasses | When this method returns, contains the minimal set of top-level class names that cover all specified concrete classes, if successful; otherwise, an empty collection. |

## Return Value

true if a covering set of top-level classes was found for all specified concrete classes; otherwise, false.

## Remarks

This method is useful for simplifying class-based queries or filters by reducing a set of concrete classes to their most general covering classes. The resulting set contains only the maximal top-level classes needed to represent the input set without redundancy.
